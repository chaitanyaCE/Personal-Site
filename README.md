# Personal-Site

Chaitanya Patel - Personal Website 🚀

Welcome to the repository for my personal portfolio website! This site serves as a digital resume and a central hub to showcase my projects, skills, and experiences as a Computer Engineering student at the University of Waterloo.

📖 Overview

This project is a clean, responsive, and accessible personal website built from scratch. It highlights my background in front-end development, UX/UI design principles, and my ongoing journey in software engineering.

Key Highlights:

Home Page (index.html): A brief introduction, personal bio, and quick links to my professional profiles.

Digital Resume (resume.html): A comprehensive, web-based version of my resume featuring my education, technical skills, projects, and experiences formatted using CSS Grid and Flexbox.

PDF Integration: A direct download link to my official PDF resume.

✨ Features

Fully Responsive Design: Adapts seamlessly to desktop, tablet, and mobile screens.

Modern UI/UX: Utilizes a clean color palette (professional navy and slate), system fonts, and subtle hover animations.

Modular CSS: Organized styling using CSS variables, Flexbox, and CSS Grid for maintainable layout structures.

Semantic HTML5: Ensures accessibility and SEO best practices.

🛠️ Technologies Used

HTML5: Page structure and semantic elements.

CSS3: Styling, layout (Grid & Flexbox), responsive media queries, and animations.

Git & GitHub: Version control and project hosting.

🚀 How to Run Locally

Since this is a static website built with pure HTML and CSS, running it locally is incredibly simple:

Clone the repository:

git clone [https://github.com/chaitanyaCE/personal-site.git](https://github.com/chaitanyaCE/personal-site.git)


Navigate to the project directory:

cd personal-site


Open the site:
Simply double-click index.html to open it in your default web browser, or use a tool like Live Server in VS Code for live reloading.
